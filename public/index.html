Step 1 — Fix the silent call issue (5 min)

  1. Go to dashboard.vapi.ai → Phone Numbers
  2. Click your number +1 (424) 699 3691
  3. Scroll to "Server URL"
  4. Paste the webhook URL above and click Save
  5. Calls will now route to your AI agent automatically

Step 2 — Get the Phone Number ID

  1. In Vapi → Phone Numbers, click your number
  2. Copy the ID shown below the number (format: 5cc793bb-85ac-4baa-…)
  3. Go to Business Setup tab, paste it into Vapi Phone Number ID
  4. Save — the backend will now match inbound calls to the right business

Step 3 — Add credits to Vapi

You currently have 10 credits (~3 min of calls). Add at least $10 for testing.
Vapi dashboard → Billing → Add credits. Claude Haiku costs ~$0.02/min, ElevenLabs TTS ~$0.01/min, Deepgram STT ~$0.01/min.
~$0.04–0.06 per minute total. $10 = ~170 minutes for testing.

Step 4 — Test a call right now

  1. Make sure Business Setup is saved with your services and hours
  2. Call +1 (424) 699 3691 from any phone
  3. The AI will answer, answer pricing questions, and book into your calendar
  4. Booked appointments appear in the Appointments tab immediately

Step 5 — Google Calendar (for real bookings)

  1. Go to console.cloud.google.com → create or select a project
  2. Enable Google Calendar API
  3. Create a Service Account → download the JSON key
  4. Copy client_email and private_key into your .env file
  5. Open Google Calendar → create a new calendar for this business
  6. Share it with the service account email (Make changes to events permission)
  7. Copy the Calendar ID → paste into Business Setup → save
